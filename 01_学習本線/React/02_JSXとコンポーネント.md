# 02_JSXとコンポーネント

## 目的

- JSXとコンポーネントの基本構造を混線せずに使えるようにする。

## ルール

- 1コンポーネント1責務
- JSXは見た目、ロジックは関数内で分離

## 要点

- JSXはJavaScript内でUI構造を書く記法
- コンポーネントは関数で定義する
- コンポーネント名は先頭大文字

## 例

```jsx
function Card() {
  return (
    <article className="card">
      <h3>Title</h3>
      <p>Body</p>
    </article>
  );
}
```

## 判断基準

1. この塊は再利用されるか
2. 子に切り出すと読みやすくなるか
3. JSX内に条件分岐を書きすぎていないか
