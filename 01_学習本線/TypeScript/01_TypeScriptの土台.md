# 01_TypeScriptの土台

## 目的

- TypeScriptを「型で事故を減らす道具」として捉える。

## ルール

- まず値の形を言語化する
- 実装より先に型の境界を決める

## 要点

- TypeScriptはJavaScriptに型システムを足したもの
- 型は実行前のチェックに使う
- 型エラーはバグの入口を早く見つけるための情報

## 例

```ts
const name: string = "Taro";
const age: number = 20;
```

## 判断基準

1. その値は何型か
2. その値は必須か任意か
3. 実行前に検出したい事故は何か
