# 03_繰り返し

## 目的

同じ処理を何度も行う基本形を整理する。

## 最初に見る結論

繰り返しは、複数のデータを順番に処理するために使う。

```php
foreach ($items as $item) {
  echo $item;
}
```

## 見るポイント

- `foreach` は配列の中身を1つずつ取り出す
- 同じHTML構造を複数回出すときに使いやすい
- WordPressのループも「投稿を順番に表示する」という意味で近い

## 自分の頭に残すこと

- 繰り返しは、複数のものを順番に処理する仕組み。
- WordPressでは投稿一覧の表示とつながる。
