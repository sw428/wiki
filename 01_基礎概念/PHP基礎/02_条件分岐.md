# 02_条件分岐

## 目的

条件によって処理を分ける基本形を整理する。

## 最初に見る結論

条件分岐は、「もしこうならこの処理、そうでなければ別の処理」を書くためのもの。

```php
if ($is_home) {
  echo 'home';
} else {
  echo 'other';
}
```

## 見るポイント

- `if` は条件が合うときの処理
- `else` は条件が合わないときの処理
- WordPressでは「このページなら」「投稿があるなら」の判断でよく使う

## 自分の頭に残すこと

- 条件分岐は処理の分かれ道。
- WordPressでは表示条件の判断に使う。
