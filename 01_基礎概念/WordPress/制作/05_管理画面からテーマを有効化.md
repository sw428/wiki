# 05_管理画面からテーマを有効化

## 目的

作ったテーマをWordPress管理画面から有効化する手順を整理する。

## 最初に見る結論

テーマファイルを作っただけでは使われない。管理画面から有効化する必要がある。

## 確認順

1. WordPress管理画面に入る
2. 「外観」を開く
3. 作ったテーマが表示されているか見る
4. テーマを有効化する
5. 表示画面で反映されているか確認する

## 表示されないとき

- テーマフォルダの場所が違う
- `style.css` に `Theme Name` がない
- `index.php` がない
- 見ているサイトが違う

## 自分の頭に残すこと

- 作ったテーマは、最後に管理画面で有効化する。
- 見えないときは `style.css` と `index.php` を確認する。
