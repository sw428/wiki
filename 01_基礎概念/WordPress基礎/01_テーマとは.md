# 01_テーマとは

## 目的

WordPressのテーマを、サイト表示を担当するファイル一式として理解する。

## 最初に見る結論

テーマは、WordPressサイトの見た目と表示ルールをまとめたもの。

## テーマに含まれるもの

- `style.css`
- `index.php`
- `functions.php`
- `header.php`
- `footer.php`
- 各種テンプレートファイル

## 混同しやすい点

テーマは1つのファイルではない。

複数のPHP / CSS / JSなどがまとまった、表示用のセットとして見る。

## 自分の頭に残すこと

- テーマはサイト表示の一式。
- 管理画面で有効化して使う。
