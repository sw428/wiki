# 04_関数

## 目的

PHPの関数を、処理に名前をつけて再利用するものとして整理する。

## 最初に見る結論

関数は、まとまった処理に名前をつけて呼び出せるようにしたもの。

```php
function show_title($title) {
  echo $title;
}

show_title('Hello');
```

## 見るポイント

- `function` で関数を作る
- 引数で外から値を渡せる
- WordPressには最初から用意されている関数が多い

## 戻り値と表示

関数には、値を返すだけのものと、関数の中で表示まで行うものがある。

値を返すだけの関数は、そのまま呼んでも画面には出ない。

```php
function get_title() {
  return 'Hello';
}

get_title();
```

表示したいときは、返ってきた値を `echo` で出す。

```php
echo get_title();
```

関数の中で `echo` している場合は、呼び出した時点で表示まで行われる。

```php
function show_title() {
  echo 'Hello';
}

show_title();
```

WordPressでは、`get_` から始まる関数は値を返すだけのことが多く、`the_` 系や `bloginfo()` は表示まで行う関数として出てくることが多い。

## 自分の頭に残すこと

- 関数は処理のまとまり。
- WordPressでは自分で作る関数と、WordPress側が用意した関数の両方を使う。
- 値を返す関数は、表示したいときに `echo` が必要。
