# 自分のVS Code環境

> [!NOTE]
> ここは、自分が必要だと思ったときにプロンプトで追記する。自動収集・自動更新は行わない。

## 目的

自分が実際に使っているVS Codeの設定・拡張機能・操作方法について、判断軸を絞って記録する。

ここはVS Code環境を完全管理する場所ではない。
記憶が抜けても、最低限どこを見ればよいか、何を採用しているか、壊れたときに何を確認するかへ戻るための場所として使う。

一般的なVS Code知識ではなく、次の内容だけを対象にする。

- 現在使っている拡張機能
- `settings.json` の採用設定
- キーボードショートカット
- Formatter・Lintの使い分け
- 環境を作り直すときの手順
- 自分の環境で起きた問題と復旧方法

## 記録ルール

- 実際に採用している設定だけを残す
- 設定値だけでなく、採用理由も短く添える
- 一般的なVS Code・Gitの知識は `04_資料庫` と分ける
- APIキー・トークン・個人情報などの秘密情報は記録しない
- 全設定・全拡張機能を細かく管理し続ける場所にはしない
- 変更した設定は、仕事に影響する判断軸だけ追える形で更新する

## 現在の環境

- OS: Windows
- エディター: VS Code
