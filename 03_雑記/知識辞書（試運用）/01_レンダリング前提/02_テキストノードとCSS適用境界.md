# 02_テキストノードとCSS適用境界

## 目的

- 「テキストノードにCSSは効かないのか？」という混線を、適用境界で切り分ける。

## ルール

- CSSセレクタが直接選べるのは**要素ノード**。
- テキストノード自体はセレクタ対象外。
- ただし、親要素に指定した文字系プロパティ（`font-size` / `color` など）は継承され、テキスト描画に反映される。

## 例

```html
<p class="lead">Hello</p>
```

```css
.lead {
  color: #334;
  font-size: 18px;
}
```

- `Hello` はテキストノードだが、親 `p.lead` の指定が反映される。
- 一方で、テキストノード単体にクラスを付けて直接指定することはできない。

## 判断基準

1. その指定は「要素を選ぶ指定」か
2. 文字として効くのは継承経由か
3. 効かない場合は親要素の適用確認に戻るか
