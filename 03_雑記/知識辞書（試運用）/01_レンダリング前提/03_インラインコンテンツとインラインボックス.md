# 03_インラインコンテンツとインラインボックス

## 目的

- 「inline と text は同じか？」を分離し、行内ズレの調査起点を揃える。

## ルール

- インライン文脈では、文字・`img`・`inline-block` などが同じ行（line box）に並ぶ。
- テキストノードは要素ではないため、要素ボックスそのものとは別扱い。
- 要素（`span` など）はインラインボックスを生成し、テキストはその文脈で描画される。

## 例

```html
<p><span class="badge">New</span> テキスト</p>
```

- `span.badge` は要素ボックスとして扱える。
- 右側の「テキスト」は要素ではなく、行内コンテンツとして配置される。

## 判断基準

1. 問題対象は要素ボックスか、テキスト描画か
2. `display` の外側文脈は inline か block か
3. ずれは `line-height` / `vertical-align` / baseline 由来か
