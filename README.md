# wiki

このリポジトリは、Drive 上のメモ・案件知識・学習内容・説明方針を、
継続的に整理・統合していくための wiki です。

司令塔は ChatGPT 側に置き、
Codex は整理・反映・修正の作業担当として使います。

この repo は、知識を置くだけの倉庫ではなく、
プロジェクト全体の状況を把握しやすくし、
判断を再利用しやすくするための外部脳として運用します。

---

## トップ階層

- `00_Docsの使い方`  
  この wiki 自体の使い方、構造ルール、運用ルール

- `01_案件・制作`  
  案件、ポートフォリオ実装、制作中の設計・判断

- `02_学習本線`  
  今まさに進行中の学習テーマ、本線メモ

- `03_雑記`  
  一時メモ、未整理メモ、思考の断片  
  迷ったらここ

- `04_資料庫`  
  辞書的に参照する内容、整理済み知識、再利用用資料

- `05_保留`  
  置き場がまだ決めきれないもの、後で見直すもの

---

## 基本方針

- 司令塔の核は ChatGPT 側に置く
- Codex に丸投げせず、作業担当として使う
- まず現在地を把握しやすい構造を優先する
- 迷うものは無理に整理せず `03_雑記` または `05_保留` に置く
- 知識量だけでなく、説明の噛み合い方・混線点・判断理由も残す
