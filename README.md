# wiki

このリポジトリは、GPTとのリアルタイム対話で生まれたメモ・案件知識・学習内容・説明方針を、Codex が最終的に整理して統合するための wiki です。

> [!IMPORTANT]
> この repo は「知識を置くだけの倉庫」ではなく、再把握しやすく、判断を再利用しやすい外部脳として運用する。

> [!NOTE]
> 司令塔の核は ChatGPT 側に置き、Codex は整理・反映・修正の実作業担当として使う。

> [!TIP]
> 運用中に `AGENTS.md` / `CURRENT.md` / `README.md` は Codex により継続的に微調整される。

---

## 自動化スクリプト

- 説明: [scripts/README.md](./scripts/README.md)
- 新規スクリプト追加時は「何をするスクリプトか」を明記し、リスクを **☆5段階** で表示する
- 個人情報に関わる表記は、可能な限り一般名へ自動で言い換えて明記するよう努める
- 法令・利用規約・権限範囲を超える操作は行わない
- 実行対象は **このプロジェクト（`<PROJECT_ROOT>`）のみ** に限定する
- 自然言語で Codex に依頼した実行でも、最終確認と実行判断は利用者側で行う
- OS操作や Git 操作を含むため、利用は **自己責任**
- 悪意のある利用は行わない
- 個人情報の一般名化はアプリ性能に依存し、取りこぼしがあり得る前提で利用者側でも確認する
- 直接利益を得ない私的活用・非商用（無料）利用を前提にする
- このツールで情報整理し、学習・能力向上や実務で間接的な付加価値を得る目的の利用は許容する
- 情報整理の結果を発信すること自体は可能とする
- その場合でも、著作権・各種権利・利用規約は適用される
- OpenAI 側の最新の規約・ポリシーで許可される範囲でのみ利用する
- 不安を感じる場合は `scripts` フォルダを削除して使用してよい
- `scripts` をすべて削除しても、wiki 本体（md運用）の基本機能は失われない
- Codex の運用性能は主に md ファイルの自然言語構造に依存し、`scripts` は補助扱い

## トップ階層

- `01_学習本線`: レベル1（基本概念）。進行中の学習テーマ、本線メモ
- `02_案件・制作`: レベル2（実務）。案件、ポートフォリオ実装、制作中の設計・判断
- `03_雑記`: 一時メモ、未整理メモ、思考の断片（迷ったらここ）
- `04_資料庫`: 辞書的に参照する内容、整理済み知識、再利用用資料
- `05_保留`: 置き場がまだ決めきれないもの、後で見直すもの

- 概念から固める場合: `01_学習本線` → `02_案件・制作`

## 基本方針

- 司令塔の核は ChatGPT 側に置く
- Codex に丸投げせず、作業担当として使う
- まず現在地を把握しやすい構造を優先する
- 迷うものは無理に整理せず `03_雑記` または `05_保留` に置く
- 知識量だけでなく、説明の噛み合い方・混線点・判断理由も残す
- 1階層目（top直下）フォルダは基盤のため固定し、新規作成 / rename / move / delete は明示指示がある時だけ行う
- 2階層目以降のファイルカテゴリは、既存追記で不足する場合に限り Codex が必要分を追加する
- 2階層目以降は、導線改善や重複解消のために Codex が rename / move を行う
- delete は階層を問わず、明示指示がある時だけ行う
