# 03_オブジェクトと関数の型

## 目的

- オブジェクト型と関数型を分けて読めるようにする。

## ルール

- 先にオブジェクトの形を定義する
- 関数は「入力型」と「出力型」で分ける

## 要点

- `type` / `interface` で構造を定義する
- 関数型は `(引数) => 戻り値` で読む

## 例

```ts
type User = {
  id: number;
  name: string;
};

function toLabel(user: User): string {
  return `${user.id}: ${user.name}`;
}
```

## 判断基準

1. 構造型を先に宣言できているか
2. 関数の戻り値型が明確か
3. 似た型を重複定義していないか
